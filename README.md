# ביצוע באמונה — דף נחיתה

דף נחיתה סטטי (HTML/CSS/JS ללא פריימוורק) לעסק "ביצוע באמונה" של שמואל, מיועד לקמפיין Google Ads. יעד ההמרה: השארת פרטים או פנייה בוואטסאפ.

## מבנה הפרויקט

```
index.html              דף הבית — כל התוכן והמבנה
css/styles.css          כל העיצוב, כולל משתני הצבע והפונטים
js/main.js              טפסים -> וואטסאפ, סליידר לפני/אחרי, לייטבוקס, קרוסלה, נגישות
assets/logo/            הלוגו
assets/images/          תמונות גלריה (לפני/אחרי) ותמונת שיתוף לרשתות (og-cover)
assets/testimonials/    תמונות ממליצים
assets/icons/           אייקוני SVG מקוריים לשירותים וליתרונות + מוטיב המותג
assets/qr/              קוד ה-QR שמוצג בפוטר
assets/fonts/           קבצי הפונטים (Heebo, Assistant) — תת-קבוצה עברית בלבד
favicon.svg             סמל הכרטיסייה בדפדפן
```

## איך מריצים מקומית

הכי פשוט: לחיצה כפולה על `index.html` — הדף נפתח ישירות בדפדפן ורוב הפיצ'רים עובדים (כולל הפונטים, ה-SVG והטפסים).

אם מותקן Node או Python על המחשב, אפשר גם להריץ שרת סטטי קטן (מדמה בצורה מדויקת יותר איך גוגל וכלי בדיקת ביצועים רואים את האתר):

```
npx serve .
```

או בפייתון:

```
python -m http.server 8000
```

## מה חובה לעדכן לפני עלייה לאוויר

### 1. מספר הוואטסאפ
בקובץ [`js/main.js`](js/main.js) בשורה הראשונה:
```js
const WHATSAPP_NUMBER = "972548015875";
```
פורמט: קידומת מדינה (972) בלי ה-0 המוביל, בלי סימן `+` ובלי מקפים.

### 2. כתובת האתר הסופית (וקוד ה-QR)
כרגע מוגדרת כתובת זמנית `https://bitsua-beemuna.co.il/` בשלושה מקומות:
- `js/main.js` — קבוע `SITE_URL` (לתיעוד בלבד)
- `index.html` — תגיות `<link rel="canonical">`, `og:url`

כשהכתובת הסופית תיקבע:
1. עדכנו את שלושת המקומות לעיל.
2. צרו קוד QR חדש שמצביע לכתובת (לדוגמה עם [qr-code-generator.com](https://www.qr-code-generator.com) או בקריאה ל-`https://api.qrserver.com/v1/create-qr-code/?size=400x400&format=svg&data=הכתובת-שלכם`).
3. שמרו את הקובץ שהתקבל בתור `assets/qr/qr-placeholder.svg` (או עדכנו את הנתיב ב-`index.html` בפוטר אם שיניתם שם).

### 3. פרטי קשר ואזור שירות
מופיעים בכמה מקומות ב-`index.html` — חפשו ועדכנו בכולם:
- `054-801-5875` (טלפון) — גם כ-`tel:+972548015875`
- `shmuelt2013@gmail.com`
- `bitsua.beemuna` (קישור פייסבוק בפוטר, כרגע כתובת זמנית) — עדכנו גם ב-JSON-LD בראש הקובץ (`sameAs`)
- "דרום, שפלה ומרכז" (אזור שירות) — מופיע גם בתוך ה-JSON-LD (`areaServed`)

## איך עורכים טקסטים

כל התוכן נמצא ישירות ב-`index.html`, מחולק לפי הערות סקשן ברורות (`<!-- ================= HERO ================= -->` וכו'). אין צורך בכלי בנייה — כל שינוי טקסט הוא עריכת HTML פשוטה.

## פונטים

הפונטים Heebo ו-Assistant מוטמעים כקבצים מקומיים (`assets/fonts/`) בתת-קבוצת תווים עברית בלבד, כדי לשמור על טעינה מהירה. אם בעתיד יתווסף תוכן לועזי מרובה שדורש את אותם פונטים גם עבור אותיות לועזיות, יש להוריד סט גופנים מלא יותר מ-Google Fonts.

## נגישות

- תפריט נגישות צף (הגדלת טקסט, ניגודיות גבוהה, הדגשת קישורים) — בפינה השמאלית התחתונה.
- הצהרת נגישות מלאה בקובץ `index.html` (רכיב `#a11y-statement-dialog`) — יש לעדכן את תאריך העדכון האחרון בהתאם למועד עליית האתר לאוויר.
